# 🎬 Movie Analytics Dashboard

Dive into the mesmerizing world of cinema with our cutting-edge Movie Analytics Dashboard! This powerful web application transforms raw movie data into stunning visualizations, revealing hidden patterns, blockbuster trends, and cinematic insights that will captivate any film enthusiast or data analyst.

## 🌟 Features

- **Interactive Dashboard**: Explore comprehensive movie statistics at a glance with our sleek, responsive interface.
- **Advanced Charts**: Visualize budget vs. revenue correlations, genre popularity trends, and top-grossing films with beautiful, interactive charts powered by Recharts.
- **Movie Management**: Upload and manage your movie database effortlessly.
- **AI-Powered Insights**: Leverage artificial intelligence to uncover deep analytics and comparisons.
- **Data Comparison**: Compare movies side-by-side to discover what makes a hit.
- **Real-time Statistics**: Get instant statistical analysis with simple-statistics library.

## 🚀 Tech Stack

- **Frontend**: Next.js 16, React 19
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **HTTP Client**: Axios
- **Statistics**: simple-statistics
- **Build Tools**: ESLint, Babel

## 🛠️ Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/your-username/movie-analytics-dashboard.git
   cd movie-analytics-dashboard
   ```

2. Install dependencies:

   ```bash
   npm install
   # or
   yarn install
   # or
   pnpm install
   ```

3. Run the development server:

   ```bash
   npm run dev
   # or
   yarn dev
   # or
   pnpm dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📖 Usage

- Navigate through the sidebar to access different sections: Dashboard, Movies, Upload, AI, Compare.
- Upload movie data via the Upload page.
- Explore visualizations on the Dashboard.
- Use AI insights for advanced analytics.
- Compare movies to find patterns and trends.
